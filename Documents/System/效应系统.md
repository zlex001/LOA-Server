# 效应系统

角色获得的临时状态效果，影响属性、行为、判定。

## 基础概念

**Buff实例**（Buff）是附加在角色身上的临时状态，具有效果值、持续时间。

**Buff标识**（Cid）是标识Buff种类的字符串值，如隐匿、流血、中毒、护盾。

**效果值**（EffectValue）是Buff作用强度的数值，用于计算最终效果。

**持续时间**（Duration）是Buff从生成到自动销毁的秒数，由生成源动态计算。

## 生成方式

**招式施展**：施展带特定效果的招式时自动生成对应Buff。

**生成规则**：
- Steal效果 → 生成隐匿Buff（效果值=技能等级，持续时间=技能等级秒）

**装备穿戴**：装备特定物品时获得持续Buff（未来扩展）。

**道具使用**：使用消耗品时获得临时Buff（未来扩展）。

**环境效果**：进入特定区域时获得环境Buff（未来扩展）。

## 叠加规则

**多实例共存**：同一角色可拥有多个相同Key的Buff实例。

**效果值累加**：计算时将所有同Key的Buff效果值求和。

**独立计时**：每个Buff实例独立计算持续时间。

**自动销毁**：持续时间耗尽时Buff实例自动从角色移除。

## 效果计算

### 隐匿系统

**隐匿系数**（ConcealmentValue）是所有隐匿Buff效果值的总和。

**应用场景**：偷窃成功率判定。

**计算公式**：
```
隐匿系数 = Σ(所有隐匿Buff的效果值)
成功率 = 光线系数 × Ratio(隐匿系数, 物品总价值)
```

**生成规则**：
```
施展"隐匿"招式 → 生成隐匿Buff
效果值 = 技能等级
持续时间 = 技能等级（秒）
```

**示例**：
```
10级偷窃技能施展"隐匿" → 生成隐匿Buff(效果值=10, 持续10秒)
再次施展"隐匿" → 新增隐匿Buff(效果值=10, 持续10秒)
隐匿系数 = 10 + 10 = 20
10秒后第一个buff消失 → 隐匿系数降为10
```

### 战斗系统（未来扩展）

**属性加成**：影响攻击、防御、敏捷等属性。

**状态控制**：眩晕、束缚、点穴等控制效果。

**持续伤害**：流血、中毒等DoT效果。

## 配置化

**Buff配置表**：详见 Documents/Design/Buff.md

**招式关联**：通过招式effects字段自动推断，详见 Documents/Design/招式.md

**效果值来源**：由生成源决定（如技能等级、装备属性）

**持续时间来源**：由生成逻辑动态计算

**广播配置**：
- interval字段：生效间隔（秒），"-"表示不定时生效，数字表示每N秒生效一次
- description字段：广播配置，格式为"Add:多语言键;Tick:多语言键;Remove:多语言键"
- 事件类型：Add（添加时）、Tick（生效时）、Remove（移除时）
- 支持灵活组合：可以只配置Add，或只配置Tick，或全部配置
- "-"表示完全不广播

**字段说明**：
- **id**：Buff唯一标识符
- **cid**：Buff字符串标识（用于配置引用）
- **interval**：生效间隔（秒），"-"表示不定时
- **description**：广播配置，多个事件用分号分隔

**配置示例**：
```
只在添加时：Add:【效应】[添加]隐匿
添加+移除：Add:【效应】[添加]隐匿;Remove:【效应】[移除]隐匿
全部事件：Add:【效应】[添加]流血;Tick:【效应】[生效]流血;Remove:【效应】[移除]流血
完全静默：-
```

## 数据结构

```
Buff实例
├── Cid: 隐匿
├── EffectValue: 10
├── Duration: 10秒
└── Parent: Life
```

**归属关系**：Buff属于Life.Content，随角色一起管理。

**持久化**：Buff为临时数据，不保存到数据库（重新登录后消失）。
