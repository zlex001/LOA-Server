# 教程系统（Tutorial）

负责新玩家教程的引导与进度管理，遵循"零强制暂停、零遮罩、零弹窗"设计理念。

**隐性引导**（Implicit）是通过空间和情境设计，让玩家自然发现操作方式，无需任何提示。设计方式：
- **唯一路径**：只有一个方向可走，玩家自然移动
- **障碍驱动**：必须解决障碍才能前进
- **资源诱导**：看到物品，自然拾取
- **视觉引导**：目标物明显、醒目

**显性引导**（Explicit）是通过视觉提示，主动告知玩家信息或引导玩家注意。第一次触发后不再重复。表现形式：
- **文本提示**：机制说明文字，玩家第一次遇到某机制时出现
- **视觉聚焦**：按钮光圈、元素高亮，引导玩家注意特定UI

**引导内容**（Guidance）是教程系统在各阶段发送给客户端的视觉提示，引导玩家注意特定目标或操作：

| 时刻 | 引导行为 |
|------|---------|
| 教程开始 | 发送沙地地图方位，引导前往沙地（SendWalkToSandHint） |
| 看见金矿 | 高亮金矿 + 高亮"前往"按钮 + 文本提示（SendExploreHint） |
| 到达金矿 | 高亮金矿 + 高亮"交互"按钮 + 文本提示（SendExploreHint） |
| 看见蜥蜴 | 高亮蜥蜴 + 高亮"前往"按钮 + 文本提示（SendExploreHint） |
| 到达蜥蜴 | 高亮蜥蜴 + 高亮"攻击"按钮 + 文本提示（SendExploreHint） |
| 探索完成 | 高亮"拾取"按钮 + 文本提示（SendCollectHint） |
| 看见石碑 | 高亮石碑 + 高亮"前往"按钮 + 文本提示（SendOfferingHint） |
| 到达石碑 | 高亮石碑 + 高亮"给予"按钮 + 文本提示（SendOfferingHint） |
| 石碑不可见 | 发送塔楼地图方位，引导前往（SendWalkToTowerHint） |

**新手区域**（Area）是位于某个城市场景角落的封闭结构。新玩家出生在新手区最深处，沿路完成教程内容后自然走入城市。

- **单出口**：区域只有一个出口通向城市主区域
- **线性路径**：内部地图排列成单向通道，玩家只能沿唯一方向前进
- **无缝衔接**：走出新手区即进入城市，无加载、无过场
- **可回访**：完成教程后随时可以回来，但没有额外奖励

**进度** - 教程通过完成标记跟踪玩家进度，每个标记对应一个具体动作：

| 完成标记 | 触发动作 | 引导内容 |
|---------|---------|---------|
| 到达沙地 | 玩家进入沙地地图 或 看见目标 | 引导前往沙地 |
| 金矿交互 | 玩家与金矿交互 | 看见→前往→到达→交互 |
| 蜥蜴击败 | 玩家击败蜥蜴 | 看见→前往→到达→攻击 |
| 物品收集 | 背包有金矿石 且 背包有生肉 | 引导拾取物品 |
| 石碑给予 | 给予石碑所需物品 | 引导前往石碑并给予 |

所有标记完成后，播放剧情并传送玩家到城市。金矿交互与蜥蜴击败无先后要求，玩家先看见哪个就先引导哪个，完成后自动引导另一个。

**教程入口**（Start）是由 `Login.CompleteLogin` 调用的方法，创建教程副本并进入教程。已完成教程的玩家自动跳过。

**转化**（Conversion）是在玩家理解核心玩法后，适时展示付费价值，促进首充和订阅转化。

| 触发条件 | 内容 | 说明 |
|----------|------|------|
| 达到5级 或 游玩2小时 | 首充优惠 | 展示首充礼包 |
| 达到10级 或 游玩8小时 | 月卡价值 | 介绍月卡权益（行为树等） |
| 月卡激活 | 行为树引导 | 引导使用付费功能 |
