# 养成系统

养成系统包含人物、技能、装备、宠物四大成长体系，通过等级提升、经验积累、装备强化、宠物培养等方式提升角色战斗力。

---

## 人物

人物通过击杀怪物获得经验，经验积累后提升等级，等级影响角色的基础属性与可挑战的内容。

### 锚点


1. **MaxLevel** = 100，最高等级，角色可达到的等级上限；

2. **CeilingMin** = 0，极限效率起点，即**Ceiling**(1) = 0；

3. **CeilingMax** = 3,600,000秒 = 1000小时，极限效率终点，即**Ceiling**(100)；

4. **CeilingExponent** = 4，凹凸参数，控制曲线形态（>1 下凸，前慢后快）；

5. **BattleDuration** = 10秒/敌人，战斗时长，基于战斗系统设计的单次战斗平均时长；

6. **ExpPerKill** = CurrentLevel × 10，击杀同等级怪物获得的经验（CurrentLevel 为角色升级前的等级）；

7. **MonthlyExp** = 2.0，月卡经验倍率，高级月卡的月卡经验相对于手动战斗的倍率（200%）；

8. **ExpBonus** = 0.5，经验加成，高级月卡的经验加成（+50%）；

9. **PremiumMultiplier** = 9，极限效率乘数，满配付费玩家相对于免费玩家的效率倍数；


### 推导结果


1. **Ceiling**，极限效率时间（单位：秒），高级月卡玩家从创建角色到达某等级的实际游戏时间（包含月卡经验和手动战斗）
   $$\text{Ceiling}(x) = \frac{\text{CeilingMax}}{(\text{MaxLevel})^{\text{CeilingExponent}} - 1} \cdot (x^{\text{CeilingExponent}} - 1)$$
   代入 **CeilingMax**=3600000, **MaxLevel**=100, **CeilingExponent**=4：
   $$\text{Ceiling}(x) = \frac{3600000}{100^4 - 1} \cdot (x^4 - 1)$$

2. **ExpMultiplier**，经验倍率，经验倍率道具的效果（消耗品，仅对击杀经验生效）
   $$\text{ExpMultiplier} = \frac{\text{PremiumMultiplier}}{1 + \text{ExpBonus}} - \text{MonthlyExp}$$
   代入 **PremiumMultiplier**=9, **MonthlyExp**=2.0, **ExpBonus**=0.5：
   $$\text{ExpMultiplier} = \frac{9}{1.5} - 2.0 = 6.0 - 2.0 = 4.0$$
   
   *验证公式：* $\text{PremiumMultiplier} = (\text{ExpMultiplier} + \text{MonthlyExp}) \times (1 + \text{ExpBonus}) = (4.0 + 2.0) \times 1.5 = 9$

3. **Floor**，免费时间（单位：秒），免费玩家24小时战斗升级到某等级的累计时间
   $$\text{Floor}(x) = \text{Ceiling}(x) \times \text{PremiumMultiplier}$$

4. **ΔFloor**，单级免费时间（单位：秒），从上一级升到当前等级所需的免费玩家时间
   $$\Delta\text{Floor}(x) = \text{Floor}(x) - \text{Floor}(x-1)$$

5. **ΔKills**，升级击杀数，从上一级升到当前等级需要击杀的同等级怪物数量（最小为1）
   $$\Delta\text{Kills}(x) = \max\left(1, \left\lfloor\frac{\Delta\text{Floor}(x)}{\text{BattleDuration}}\right\rfloor\right)$$

6. **ΔExp**，升级经验，从上一级升到当前等级需要的经验值
   $$\Delta\text{Exp}(x) = \Delta\text{Kills}(x) \times (x-1) \times 10$$

7. **Exp**，累计经验，从1级升到当前等级的总经验值
   $$\text{Exp}(x) = \sum_{i=1}^{x} \Delta\text{Exp}(i)$$

8. **MonthlyRate**，月卡经验速率（单位：经验/秒），高级月卡玩家每秒获得的月卡经验
   $$\text{MonthlyRate}(x) = \frac{\Delta\text{Exp}(x)}{\Delta\text{Floor}(x)} \approx x - 1$$

### Monthly Exp

**MonthlyExp**（月卡经验）是月卡持有者的被动经验收益，按真实时间持续累积，与玩家是否在线无关。

**累积机制**：
- 累积公式：$\text{AccumulatedExp} = \text{MonthlyRate}(level) \times \text{MonthlyExpRate} \times \Delta t$
- **MonthlyExpRate**：基础月卡为0.8（80%），高级月卡为1.2（120%），叠加为2.0
- $\Delta t$：自上次计算以来经过的秒数（离线期间也计入）

**计算时机**：
- 玩家登录时：计算离线期间累积的月卡经验
- 玩家领取时：计算自上次计算以来累积的经验并加到人物经验

**示例**：高级月卡玩家（50级），离线24小时后登录
$$\text{Gain} = 49 \times 2.0 \times 86400 = 8,467,200 \text{ exp}$$

4. **CeilingExponent** = 4，凹凸参数，控制曲线形态（>1 下凸，前慢后快）；

### 推导结果


1. **Ceiling**，极限效率时间（单位：秒），高级月卡玩家从学习技能到达某等级的实际游戏时间（包含月卡经验和手动使用）
   $$\text{Ceiling}(x) = \frac{\text{CeilingMax}}{(\text{MaxLevel})^{\text{CeilingExponent}} - 1} \cdot (x^{\text{CeilingExponent}} - 1)$$
   代入 **CeilingMax**=3600000, **MaxLevel**=100, **CeilingExponent**=4：
   $$\text{Ceiling}(x) = \frac{3600000}{100^4 - 1} \cdot (x^4 - 1)$$

2. **ExpMultiplier**，经验倍率，经验倍率道具的效果（消耗品，仅对使用经验生效）
   $$\text{ExpMultiplier} = \frac{\text{PremiumMultiplier}}{1 + \text{ExpBonus}}$$
   代入 **PremiumMultiplier**=9, **ExpBonus**=0.5：
   $$\text{ExpMultiplier} = \frac{9}{1.5} = 6.0$$
   
   *验证公式：* $\text{PremiumMultiplier} = \text{ExpMultiplier} \times (1 + \text{ExpBonus}) = 6.0 \times 1.5 = 9$

3. **Floor**，免费时间（单位：秒），免费玩家24小时使用升级到某等级的累计时间
   $$\text{Floor}(x) = \text{Ceiling}(x) \times \text{PremiumMultiplier}$$

4. **ΔFloor**，单级免费时间（单位：秒），从上一级升到当前等级所需的免费玩家时间
   $$\Delta\text{Floor}(x) = \text{Floor}(x) - \text{Floor}(x-1)$$

5. **ΔUses**，升级使用数，从上一级升到当前等级需要使用技能的次数（最小为1）
   $$\Delta\text{Uses}(x) = \max\left(1, \left\lfloor\frac{\Delta\text{Floor}(x)}{\text{UseDuration}}\right\rfloor\right)$$

6. **ΔExp**，升级经验，从上一级升到当前等级需要的经验值
   $$\Delta\text{Exp}(x) = \Delta\text{Uses}(x) \times \text{ExpPerUse} = \Delta\text{Uses}(x)$$

7. **Exp**，累计经验，从1级升到当前等级的总经验值
   $$\text{Exp}(x) = \sum_{i=1}^{x} \Delta\text{Exp}(i)$$

## 装备

装备通过附魔获得属性加成，一件装备只能拥有一条附魔效果，附魔后不可覆盖、不可洗练。

### 附魔效果

附魔效果分为三类共21种：

| 类别 | 效果 |
|------|------|
| 七维 | Hp、Atk、Def、Agi、Mp、Ine、Con |
| 魔法 | 水系伤害、火系伤害、风系伤害、地系伤害、水系抵抗、火系抵抗、风系抵抗、地系抵抗 |
| 制御 | 中毒命中、中毒抵抗、混乱命中、混乱抵抗、石化命中、石化抵抗 |

**七维**：直接增加角色对应属性的固定值。

**魔法**：伤害类增加角色释放对应元素魔法时造成的伤害固定值；抵抗类减少角色受到对应元素魔法时承受的伤害固定值。

**制御**：命中类增加角色释放对应异常状态魔法时的命中率；抵抗类增加角色对应异常状态的抵抗率。

## 宠物

宠物作为Life，养成机制与人物相同：通过击杀怪物获得经验，经验积累后提升等级。

### 经验加成

宠物经验与人物共享月卡经验和经验加成，额外受驯兽技能加成影响：

| 加成来源 | 最大倍率 | 说明 |
|---------|:-------:|------|
| 月卡经验 | ×2 | 高级月卡，与人物共享 |
| 经验加成 | +50% | 高级月卡，与人物共享 |
| 驯兽技能 | ×10 | 技能等级越高，宠物经验加成越高 |
| 宠物经验倍率 | ×4 | 商城道具，按次数消耗 |

**极限效率**：PremiumMultiplier = 人物极限效率 × 驯兽技能加成 = 9 × 10 = 90

**驯兽技能加成**：

$$
\text{技能加成} = 1 + 9 \times \frac{\text{驯兽技能等级}}{\text{MaxLevel}}
$$

*驯兽技能1级时加成×1.09，100级时加成×10。*
