# 新手引导系统

**新手引导系统**（Tutorial System）负责为新玩家提供触发式引导，帮助玩家理解核心玩法、完成生存循环、感知付费价值。

本系统采用服务端驱动的触发式设计，根据玩家状态和行为触发相应引导，通过协议推送给客户端展示。

---

## 体验设计 | NPE

**NPE**（New Player Experience，新玩家体验）定义新玩家前7天的体验旅程，确保玩家理解核心玩法、体验游戏价值、到达转化节点。

### 生命周期概览

| 阶段 | 时长 | 目标 | 关键指标 |
|------|------|------|----------|
| **Survival** | 0-2小时 | 理解核心生存循环 | 引导完成率 |
| **Discovery** | 2-8小时 | 体验战斗与市场 | D1留存 |
| **Conversion** | 第2-3天 | 首充决策 | 首充率 |
| **Mastery** | 第4-7天 | 核心循环掌握 | D7留存 |
| **Commitment** | 第7天+ | 月卡订阅 | 付费率 |

**0-0.5小时：首次接触**

| 触发条件 | 首次遭遇敌人 |
| 引导类型 | 上下文提示 |

## 协议设计 | Protocol

### 现有协议

当前 `Net.Protocol.Tutorial` 仅发送一个ID：

```csharp
public class Tutorial : Base
{
    public int id;
    public Tutorial(int id) { this.id = id; }
}
```

### 扩展协议

为支持 **NPE** 设计，需要更丰富的引导信息：

**服务端→客户端：Guidance协议**

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| id | int | 是 | 唯一引导ID，用于跟踪 |
| type | int | 是 | **GuidanceType** 枚举值 |
| title | string | 否 | 本地化标题 |
| content | string | 是 | 本地化内容 |
| action | object | 否 | 操作按钮配置 |
| waypoint | int[] | 否 | 地图坐标 [x, y, z] |

**GuidanceType** 枚举：

| 值 | 名称 | 说明 |
|----|------|------|
| 0 | SystemPrompt | 模态对话框，需要关闭 |
| 1 | ContextualHint | 非阻塞浮动文本 |
| 2 | Waypoint | 地图上的可视标记 |
| 3 | Achievement | 奖励通知弹窗 |
| 4 | SoftPromotion | 价值主张展示 |
| 5 | Tutorial | 交互式分步指引 |

**客户端→服务端：GuidanceResponse协议**

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| id | int | 是 | 响应的引导ID |
| status | int | 是 | 响应状态枚举值 |

**Status** 枚举：

| 值 | 名称 | 说明 |
|----|------|------|
| 0 | Completed | 玩家完成/确认引导 |
| 1 | Dismissed | 玩家手动关闭 |
| 2 | ActionTaken | 玩家点击操作按钮 |

## 状态管理 | State

### 玩家引导状态

```csharp
public class PlayerGuidanceState
{
    // 已完成的一次性触发
    public HashSet<int> Completed { get; set; } = new();
    
    // 冷却跟踪：引导ID -> 上次显示时间
    public Dictionary<int, DateTime> LastShown { get; set; } = new();
    
    // 玩家主动关闭（不再显示）
    public HashSet<int> Dismissed { get; set; } = new();
}
```

### 触发检查逻辑

```csharp
public bool CanTrigger(Player player, int guidanceId, GuidanceConfig config)
{
    var state = player.GuidanceState;
    
    // 检查是否已完成（一次性）
    if (config.OneTime && state.Completed.Contains(guidanceId))
        return false;
    
    // 检查是否已关闭
    if (state.Dismissed.Contains(guidanceId))
        return false;
    
    // 检查冷却
    if (config.Cooldown > 0 && state.LastShown.TryGetValue(guidanceId, out var last))
    {
        if (DateTime.Now - last < TimeSpan.FromSeconds(config.Cooldown))
            return false;
    }
    
    // 检查条件
    return config.Condition(player);
}
```

## 客户端需求 | Client

### 显示组件

| GuidanceType | UI组件 | 行为 |
|--------------|--------|------|
| SystemPrompt | 模态对话框 | 阻塞输入，需要操作 |
| ContextualHint | 浮动文本 | 5秒后自动消失，非阻塞 |
| Waypoint | 地图标记 + 箭头 | 到达或关闭后消失 |
| Achievement | 弹出横幅 | 3秒后自动消失 |
| SoftPromotion | 底部面板 | 可关闭，有操作按钮 |
| Tutorial | 步骤覆盖层 | 高亮UI元素，等待操作 |
